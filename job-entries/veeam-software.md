Developed internal automation tools that improve the workflow of engineers working on the main company product - Veeam Backup & Replication (VBR).

**Technologies:** C#, .NET, Aspire, Hangfire, PowerShell, Azure DevOps, Git, WSL

### Highlights

- Designed and built LogManager, a command-line tool that lets developers and QA search and analyze big archives of application logs generated by VBR. The tool uses asynchronous processing to fasten processing and keep searches responsive even on very large datasets.

    - Owned LogManager from start to finish, presented the application within the company and gathered feedback.

- Developed a tool that visualizes NuGet dependencies across a large multi-repository codebase and helps developers automate package updates.

- Automated adoption of new .NET tooling across repositories, including migration to the new solution format (*.slnx) and the transition from VSTest to Microsoft Testing Platform (MTP).

- Added Microsoft Teams integration to an internal Azure DevOps notification system, researching Microsoft delivery systems and working with the security team to define the required permissions.

- Built automation for running VBR for Linux inside Windows Subsystem for Linux (WSL), researching the platform and creating PowerShell scripts that made local testing significantly faster.
