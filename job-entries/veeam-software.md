Developed internal tools and automation that improve the workflow of engineers working on Veeam Backup & Replication (VBR).

**Technologies:** C#, .NET, PowerShell, Azure DevOps, Git, WSL, Async Programming, CLI Development

### Highlights

- Designed and built LogManager, a command-line tool that lets developers and QA search and analyze hundreds of gigabytes of application logs generated by Veeam Backup & Replication. The tool uses asynchronous processing to keep searches responsive even on very large datasets.
- Owned LogManager from start to finish, presented the application within the company and gathered feedback.
- Built automation for running VBR for Linux inside Windows Subsystem for Linux (WSL), researching the platform and creating PowerShell scripts that made local testing significantly faster.
- Developed a tool that visualizes NuGet dependencies across a large multi-repository codebase and helps developers automate package updates.
- Automated adoption of new .NET tooling across repositories, including migration to the new solution format (*.slnx) and the transition from VSTest to Microsoft Testing Platform (MTP).
- Added Microsoft Teams integration to an internal Azure DevOps notification system, working with the security team to define the required Microsoft Graph permissions.
